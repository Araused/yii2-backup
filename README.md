Yii2-backup
===================
Basic Yii2 site backup methods

Installation
---
Run
```code
php composer.phar require "demi/backup" "~1.0"
```
or


Add to composer.json in your project
```json
{
	"require": {
  		"demi/backup": "~1.0"
	}
}
```
then run command
```code
php composer.phar update
```
